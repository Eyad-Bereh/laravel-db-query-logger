# Installation

### Requirements

Before installation, please confirm your system meets these minimum requirements:

* PHP ≥ 8.1
* Laravel Framework ≥ 10.0

### Installation

The package is installed using [Composer](https://getcomposer.org/), the dependency manager for PHP.

1. Open your terminal.
2. Navigate to your Laravel project's root directory.
3. Run the following command to download and install the package:

```bash
composer require eyad-bereh/laravel-db-query-logger
```
